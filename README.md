## 🔬 Projets Phares

### 1. Agent IA Autonome - Suivi des Éruptions Solaires (NASA API)
*Démonstration d'orchestration de LLM avec Tool Calling.*
* **Automatisation :** Workflow complet développé sur **n8n**.
* **Architecture IA :** Agent conversationnel **LangChain** utilisant l'API **Google Gemini**.
* **Intégration API :** Configuration de l'agent pour extraire et synthétiser dynamiquement des données brutes JSON depuis la base de données de la NASA (DONKI).

### 2. Pipeline Industriel d'Analyse de Sentiments (ABSA)
*Conception d'un pipeline complet pour extraire et classifier les aspects/sentiments de **20 000 avis clients**.*
* **NLP & Extraction :** Nettoyage textuel et extraction des thématiques via l'algorithme LDA.
* **Fine-Tuning pour la production :** Ajustement d'un modèle **XLM-RoBERTa** pour la classification finale.
* **Ingénierie MLOps :** Déploiement complet via une API **FastAPI** et conteneurisation **Docker** pour garantir la stabilité en production.

### 3. Assistant Hôtelier Hybride (RAG & Fine-Tuning)
*Développement d'un agent fiable avec contrôle des hallucinations.*
* **Architecture :** Système **RAG** (Retrieval-Augmented Generation) avec LangChain et la base vectorielle FAISS.
* **Optimisation :** Fine-tuning de **Llama-3-8B** avec la méthode LoRA/Unsloth pour forcer l'alignement du modèle sur un ton de service client strict.
